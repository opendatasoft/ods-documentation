Configuring Blocks
==================

Blocks contain editable content.
They are displayed in the different sections on your page.

There are three types of blocks available in the Studio:

- :doc:`Chart <add-chart>`
- :doc:`KPI <add-kpi>`
- :doc:`Text <add-text>`

To start with the step-by-step block configuration, click the desired block on the preview pane.

.. admonition:: Note
   :class: note

   If you do not configure a block, it will leave a blank space on your page.

When configuring a chart or a KPI, you can also :doc:`refine the data to be displayed <refine-data>`.

You can easily position blocks on the page using the layout editor.
For more information about positioning blocks on your page, see :ref:`editing-page-layout`.
